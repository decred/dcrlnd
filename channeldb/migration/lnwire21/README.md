lnwire
======

[![Build Status](http://img.shields.io/travis/decred/dcrlnd.svg)](https://travis-ci.org/decred/dcrlnd) 
[![MIT licensed](https://img.shields.io/badge/license-MIT-blue.svg)](https://github.com/decred/dcrlnd/blob/master/LICENSE)
[![GoDoc](https://img.shields.io/badge/godoc-reference-blue.svg)](http://godoc.org/github.com/decred/dcrlnd/lnwire)

The lnwire package implements the Lightning Network wire protocol.

This package has intentionally been designed so it can be used as a standalone
package for any projects needing to interface with lightning peers at the wire
protocol level.

## Installation and Updating

```bash
$ go get -u github.com/decred/dcrlnd/lnwire
```
